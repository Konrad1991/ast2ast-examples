Welcome to the ast2ast examples page

This page presents a series of tutorials demonstrating how to use the ast2ast package in conjunction with r2sundials and paropt to solve ordinary differential equations (ODEs) and optimize their parameters. Each tutorial focuses on a different aspect of these packages, providing a comprehensive overview of their capabilities.

r2sundials Tutorial

This tutorial demonstrates how to use r2sundials to solve ODEs. It covers the basics of defining an ODE system, setting up the solver, and interpreting the results.

Open r2sundials Tutorial

PSO Tutorial

This tutorial focuses on the use of the Particle Swarm Optimization (PSO) algorithm for parameter optimization. It explains how PSO works, how to set it up in paropt, and how to interpret the results.

Open PSO Tutorial

Paropt Tutorial

This tutorial provides a comprehensive overview of the paropt package. It covers the process of defining an ODE system, translating it to C++, solving it with Sundials, and optimizing the parameters with PSO. It also explains the advanced features of paropt, such as memory borrowing and parallel computing.
